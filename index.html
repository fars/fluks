fluks by markuspeloquin

Free LUKS, a free (as in non-GPL) implementation of Linux Unified Key Setup, written in C++.

Dependencies

Compile time: Boost.Build Runtime: Boost, OpenSSL, Device-mapper

Install

Run 'bjam'.
